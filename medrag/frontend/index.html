MedRAG 医学文献问答系统
Conversation API

上传医学文献，然后开始提问

左侧负责上传、索引和检索调试，中间负责显示问答结果，底部负责输入问题。 PDF 索引完成后会创建 conversation，连续提问会保存 user、assistant 消息和引用来源。

1. 建立索引 先在左侧上传 PDF，得到 document_id。
2. 输入问题 底部输入医学文献相关问题，设置 top-k。
3. 查看引用 回答下方会显示检索到的原文片段。

本系统仅用于医学文献辅助阅读，不提供临床诊断或治疗建议。
